¿Qué es un Computador?

Un computador es una máquina electrónica que procesa datos y los convierte en información útil. Su funcionamiento se basa en dos conceptos fundamentales:

Hardware

  • CPU: Cerebro del computador
  • Memoria RAM: Memoria de trabajo temporal
  • Disco Duro: Almacenamiento permanente
  • Periféricos: Dispositivos de entrada/salida

Software

  • Sistema Operativo: Windows, Linux, macOS
  • Aplicaciones: Word, Excel, Navegadores
  • Utilidades: Antivirus, Compresores
  • Lenguajes: Python, Java, JavaScript
